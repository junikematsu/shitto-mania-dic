これは、婦人公論.jpで連載されたエッセイ『嫉妬マニア』（著者：斉藤ナミ）の記事18本を元に作成した、AI向けの「嫉妬プロフィール帳」データセットです。本データセットの作成は、著者の斉藤ナミさんとの共同作業です。
